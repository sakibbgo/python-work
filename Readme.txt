- Install Python 2.7 or higher, if not already installed
- Put the PilotDownloadAndInstall.py on a folder you like
- shift+mouse right click and select: Open command window here

Run the command like below:

python PilotDownloadAndInstall.py 8.2 PDS where first arguent "8.2" is version number and the second argument "PDS" is product name you would like to install. Only two product supported now: Pilot and PDS (The names are case sensitive when you run the script)

To install Pilot type Pilot instead PDS and to install trunk type 0.0 instead 8.2


You can schedule a job with this script, for daily pilot update.
